Recent Projects

Nina Williams Jewelry

NinaWilliamsJewelry.com is a database-driven e-commerce website. My client requested an online store to showcase a series of handmade jewelry collections. I built this site using PHP and MySQL on the server to organize and populate data dynamically.

While the site does feature PayPal integration, it does not use any frameworks or plugins. Like all of the websites and applications that I develop, NinaWilliamsJewelry.com has been fully optimized for desktop, tablet and mobile users.

Jeremy Cochran Guitars

My client requested a simple static site to feature his collection of custom cookie tin guitars. I used primarily HTML5, CSS3 and Javascript to build this site. The homepage features an excellent JQuery image slider plugin (Cycle2).

In this case my client wanted to keep it simple- a gallery of guitars with individual pages that include descriptions, images and pricing details for each item. JeremyCochranGuitars.com is fully optimized for desktop, tablet and mobile users.
